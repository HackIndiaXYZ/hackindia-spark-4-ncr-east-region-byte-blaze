# 🌾 InsuChain

> A blockchain-powered crop insurance platform that provides automated, transparent, and fair insurance for farmers using real-time weather data.

---

## 📌 Problem Statement

Farmers face multiple challenges with traditional crop insurance systems:

- ❌ Delayed claim processing  
- ❌ Lack of transparency  
- ❌ Manual verification leads to corruption  
- ❌ Difficulty in proving crop damage  
- ❌ No real-time data integration  

This results in financial loss and lack of trust in insurance systems.

---

## 💡 Solution

**InsuChain** is a decentralized crop insurance platform that leverages **blockchain + weather APIs** to automate the entire insurance lifecycle.

- 🌦️ Real-time weather data tracking  
- 📜 Smart contract-based policy management  
- 💸 Automatic claim settlement  
- 🔗 Transparent transactions on blockchain  
- 🧾 Immutable record keeping

## ⚙️ Workflow

1. **User Registration & Login**
   - Farmers register/login via frontend  
   - Secure authentication using JWT  

2. **Policy Purchase**
   - Farmer selects crop type & insurance plan  
   - Policy details stored in backend + blockchain  

3. **Weather Monitoring**
   - System continuously fetches weather data  
   - Conditions like drought / heavy rain are tracked  

4. **Smart Contract Trigger**
   - If weather conditions meet claim criteria  
   - Smart contract automatically triggers payout  

5. **Automatic Claim Settlement**
   - No manual verification needed  
   - Funds transferred securely  

6. **Blockchain Transparency**
   - All transactions recorded on Polygon  
   - Publicly verifiable
